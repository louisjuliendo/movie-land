## Key Features

- Search for movies
- Display movie information (movie name, image, release date, rating, and overview)

## Run Project Locally

- git clone https://github.com/louisjuliendo/movie-land.git
- create .env file and add REACT_APP_MOVIE_API_KEY=Y0UR4P1K3Y
- npm install
- npm run start
